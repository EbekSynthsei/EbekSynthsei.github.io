Emanuele's Portfolio Website

A modern, responsive portfolio website for showcasing professional skills and projects

Project Overview

This portfolio website was designed for Emanuele to showcase his programming skills, projects, and professional experience. The website features a modern, responsive design with a cobalt blue and gold color scheme, interactive elements, and a user-friendly interface that works seamlessly across all devices.

Key Features

  • Responsive design that adapts to all screen sizes
  • Interactive project showcase with filtering capabilities
  • Dark/light mode toggle for user preference
  • Smooth scrolling navigation and animations
  • Contact form with validation
  • Optimized for performance and accessibility
